# Kinetic Gain Embedded — tie-back

## What this repo demonstrates

This repo demonstrates the **audit stream** primitive — consent capture tied to policy versions, revocation events, downstream reset blockers, and audit-safe operational visibility.

## How Kinetic Gain Embedded extends it

Kinetic Gain Embedded is security-first embedded analytics for B2B SaaS.
It extends this primitive to consent-aware, policy-aware, and audit-aware in-product analytics for digital health, privacy operations, and patient workflow products, wrapped in:

- signed JWT authentication
- semantic-layer row-level security (RLS)
- read-only data connectors
- AST SQL validation
- append-only audit events
